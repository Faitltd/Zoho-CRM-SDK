# Express Zoho CRM Webhook Server

This template demonstrates:
- Express webhook handler with signature verification
- SDK middleware attachment
- Error handling and rate limiting

## Setup
1. Copy into a Node/Express project.
2. Install dependencies:
   `npm install @yourcompany/zoho-crm @yourcompany/zoho-crm-express`
3. Configure env vars:
   `ZOHO_CLIENT_ID`, `ZOHO_CLIENT_SECRET`, `ZOHO_REFRESH_TOKEN`, `ZOHO_REGION`, `ZOHO_WEBHOOK_SECRET`
